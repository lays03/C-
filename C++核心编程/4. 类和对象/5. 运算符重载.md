## 4.5 运算符重载
对已有的运算符重新进行定义，赋予其另一种功能，以适应不同的数据类型

### 4.5.1 加号运算符重载
作用：实现两个自定义数据类型相加的运算
通过成员函数重载加号
```cpp
Person Person::operator+(Person &p)
{
    Person temp;
    temp.m_A = this->m_A + p.m_A;
    temp.m_B = this->m_B + p.m_B;
    return temp;
}

Person p1;
p1.m_A = 10; p1.m_B = 10;
Person p2;
p2.m_A = 10; p2.m_B = 10;
Person p3 = p1 + p2; //本质为：p3 = p1.operator(p2);

```

总结1：对于内置的数据类型的表达式的运算符是不可能改变的（比如不能重载使得 1+1=0）<br>
总结2：不要滥用运算符重载
